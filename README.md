# lifegame
Conway's Game of Life

## RULE(23-3)
* 해당 셀이 살아있을 때 인접한 셀 중 살아있는 셀이 2~3 개면 다음 세대에도 살아남는다.
* 죽은 셀 주위에 살아있는 셀이 3개면 다음세대에 살아난다.
* 그 외에는 다음세대에 죽는다(인접한 셀 중 살아있는 셀이 1개이거나 4개이상일때)
